# 一行代码实现RecyclerView
Step 1. Add the JitPack repository to your build file

    allprojects {
     	 repositories {
		       maven { url 'https://jitpack.io' }
		       
	 	}
    }
  
Step 2. Add the dependency

    dependencies {
    	        implementation 'com.github.zihuan1:view-grace-recyclerview:last.integration'
	}
